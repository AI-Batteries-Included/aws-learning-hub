# MDX Test Page

Welcome to our **MDX test page**! This page demonstrates the integration of Markdown content with React components in Next.js.

## What is MDX?

MDX is a powerful tool that allows you to write JSX embedded inside Markdown. This enables you to:

- Write content in Markdown format
- Embed interactive React components
- Combine the best of both worlds

### Features Demonstrated

Here are the key features we're testing:

1. **Markdown formatting** - Headers, paragraphs, lists
2. **Code blocks** - Both inline `code` and block code
3. **React components** - Interactive elements
4. **TypeScript support** - Full type safety
